# Unity MLAgents Robot Arm

## Overview
This project implements a multi-joint robotic arm in Unity, designed for experimentation with **Inverse Kinematics (IK)** and **Machine Learning (ML-Agents)**.

## Features
- 10-bone robot arm hierarchy  
- 6-axis controllable joints  
- Manual joint control for testing  
- Ready for Unity ML-Agents training  
- Suitable for IK and reinforcement learning experiments
